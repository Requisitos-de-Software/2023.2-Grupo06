# Léxicos

## Introdução
A técnica de léxicos visa estabelecer uma linguagem comum entre os envolvidos no desenvolvimento do software, eliminando ambiguidades e garantindo uma compreensão unificada dos requisitos. Ela faz isso descrevendo símbolos de uma linguagem: o principal objetivo a ser perseguido pelos Engenheiros de Requisitos é identificar palavras ou frases específicas do meio social da aplicação que está sendo estudada <a id="a" href="#aa">[1]</a>.

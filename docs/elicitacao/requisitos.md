# Requisitos Elicitados
Na tabela 1 é possível identificar todos os requisitos elicitados e na legenda pode-se verificar a rastreabilidade:

Legenda:

- RF - Requisito Funcional
- RNF - Requisito Não Funcional
- OB - Observação
- IN - Introspecção
- BS - Brainstorming
